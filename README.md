# GUARDIAN · 守护者智能项圈

> 好看，只是它的开始。
> *It starts with style. It stays for life.*

GUARDIAN 是一款「潮流外衣 × 守护内核」的宠物智能项圈概念项目：以可换壳的高颜值设计为入口，内置实时定位与电子围栏，并通过 GUARDIAN+ 会员，基于 365 天连续数据为每一只宠物建立专属的生理基准线——*提醒，不打扰；守护，不缺席。*

本仓库为该项目的**对外展示站点**(品牌官网 + App 高保真原型 + 发布会影片)。

## 🌐 在线访问

部署于 GitHub Pages，构建后可访问：

- **品牌官网** — 站点首页
- **App 原型** — `/app/`
- **发布会影片** — `/launch-film.mp4`
